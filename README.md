# Bulma Extensions

A collection of simple extensions for [Bulma 1.0.x](https://bulma.io), re-designed and updated for modern web standards.

## Components

| Component | Summary | Status |
| :--- | :--- | :--- |
| [divider](packages/divider) | A Divider Stylesheet for separating content | Ready |
| [timeline](packages/timeline) | A Timeline Stylesheet for displaying chronological events | Ready |
| [pageloader](packages/pageloader) | A PageLoader / PageBlocker Stylesheet for loading states | Ready |
| [ribbon](packages/ribbon) | A Ribbon Stylesheet for displaying tags or labels on containers | Work-In-Progress |
| [badge](packages/badge) | A Badge Stylesheet for displaying counters or status icons | Work-In-Progress |
| [accordion](packages/accordion) | An Accordion Stylesheet for collapsible content sections | Work-In-Progress |
| [carousel](packages/carousel) | A Carousel Stylesheet for cycling through content | Draft |
| [pricingtable](packages/pricingtable) | A PricingTable Stylesheet for displaying product plans | Draft |

## Installation & Usage

Each package is managed individually. Please refer to the specific package directory in `packages/` for detailed installation and usage instructions.

Generally, you can install a package using pnpm:

```bash
pnpm add @bulma-ext/<package-name>
```

And include it in your project after the main Bulma stylesheet.
